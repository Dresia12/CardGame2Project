Ce dossier contient les fichiers sonores utilisés par l'interface du jeu CardGame2.

Fichiers attendus (format .wav, mono, court) :
- attack.wav         : joué lors d'une attaque
- heal.wav           : joué lors d'un soin
- effect_expire.wav  : joué lors de l'expiration d'un effet
- victory.wav        : joué lors d'une victoire
- defeat.wav         : joué lors d'une défaite

Placez ici vos propres sons ou utilisez des sons libres de droits.
